# 中国上市银行数据库

38 家 A 股上市银行全景看板

## 数据

| 分类 | 数量 |
|------|------|
| 国有大行 | 6 家 |
| 股份制银行 | 9 家 |
| 城商行 / 农商行 | 23 家 |
| **合计** | **38 家** |

## 文件

| 文件 | 说明 |
|------|------|
| `index.html` | 离线看板（内嵌数据，双击即可查看） |
| `banks.json` | 银行数据 JSON（供 index.html 使用） |
| `foreign_banks.json` | 外资银行数据（12家，含在华分支信息） |
| `download_bank_data.py` | 数据补充脚本（补全名称、总行地址、财务指标） |

## 看板功能

- 按银行类型筛选（国有大行 / 股份制 / 城商行农商行）
- 按城市筛选
- 关键词搜索（银行名称、代码、全称）
- 每张卡片展示：名称、全称、代码、类型、总行/注册地址、官网、交易所、板块、上市日期

## 补充数据

网络通畅时运行：

```bash
cd bank/
python3 download_bank_data.py
```

将自动补全：公司全称、总行地址、官网，以及财务指标（总资产、净利润、市值）。

数据源：akshare（cninfo + Eastmoney）
