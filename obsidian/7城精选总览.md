# 📊 7城上市公司精选总览

> 深圳 · 广州 · 佛山 · 珠海 · 福州 · 厦门 · 长沙 — 共 {len(companies)} 家

## 🏙️ 按城市

```dataview
TABLE length(rows) as "数量"
FROM "companies"
GROUP BY city
SORT rows.length DESC
```

## 📋 全部企业

```dataview
TABLE stock_code as "代码", city as "城市", board as "板块", industry as "行业", match_type as "匹配"
FROM "companies"
SORT city ASC, industry ASC
```

## 🏦 上市银行

```dataview
TABLE stock_code as "代码", city as "城市", bank_type as "银行类型", swift_code as "SWIFT"
FROM "companies"
WHERE is_bank = true
SORT city ASC
```
